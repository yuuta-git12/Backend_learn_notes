#PHP 
- PHP依存関係管理ツール
- PHPやLaravelのアプリケーションに必要なライブラリがある場合、Composerを通してインストールすると、依存関係を解決してくれる
	- ライブラリAを動作させるために必要なライブラリを合わせてインストールしてくれる

## composer.json
- プロジェクトの設定、ライブラリの依存関係が記述されたファイル

## composer.lock
- 実際にインストールするライブラリとそのバージョンを記録している
- プロジェクトのメンバーと共有することで、全員同じライブラリをインストールすることができる


## ライブラリのアップデート方法
- `composer.json` の `require-dev` にあるライブラリのバージョン制約を変更
```php
// 制約を変更後　以下を実行
composer update [ライブラリ名]
```
