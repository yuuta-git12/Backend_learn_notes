#Laravel 
```
── Laravelアプリケーション
 ├── app/ # アプリケーションロジック用フォルダ
 ├── bootstrap/ # 起動コード用フォルダ(基本的に触らない)
 ├── config/ # Laravel設定ファイル
 ├── database/ # マイグレーション・シーダーなどのDB関連ファイル用フォルダ
 ├── resources/ # ビュー・アセット用フォルダ
 ├── routes/ # ルーティングファイル用フォルダ
 ├── storage/ # ファイルストレージ（ログなどのLaravelが生成するファイル用フォルダ）
 ├── tests/ # テストコード用フォルダ
 ├── vendor/ # Composerでインストールしたライブラリ用フォルダ

```
