#Laravel 
# コントローラーの作成
```
php artisan make:controller [コントローラー名]
// または
sail artisan make:controller [コントローラー名]
```
- 生成されるファイル
	- `app/Http/Controllers/コントローラ名.php`

## CRUD操作のメソッドを自動生成したい場合
```
php artisan make:controller [コントローラー名] --resource
// または
sail artisan make:controller [コントローラー名] --resource
```
- 生成されるファイル
	- `app/Http/Controllers/コントローラ名.php`
		- ndex, create, store, show, edit, update, destroyメソッド付き
