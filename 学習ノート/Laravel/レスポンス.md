- サーバーからクライアントに返却される情報

# Laravelの場合
- コントローラーのメソッドの戻り値がレスポンスとしてクライアントに返却される
## Laravelのレスポンスの種類
### JSON形式
- Eloquentのモデル、コレクションを返却するとJSON形式に変換する
- レスポンスオブジェクトの`json`メソッドを使用した場合もJSON形式で返却可能
```php
return response()->json([
	'name' => 'Yuta',
	'age' => '20',
]);
```
### ビュー
- ビューの返却は`view`メソッドの第1引数にビューファイルのパス、第2引数にビューに渡したい連想配列の形で渡す。
	- 連想配列のキー名で、ビュー側は値を参照できる
```php
public function create(): View {

$categories = Category::all();

// ビューオブジェクトを返す
return view('admin/book/create', ['categories' => $categories]);
}
```
### リダイレクト
- ブラウザに、瞬時に新しいリクエストを送信させる命令
	- データベースを更新するリクエストなどの「再送してほしくないリクエスト」の後はリダイレクトを実行することで、最後に送ったリクエストを上書きする。r

# レスポンスオブジェクト
- ヘッダーなど詳細な情報を設定する場合はレスポンスオブジェクトを使用する必要がある
- レスポンスオブジェクトをは`response`メソッドで取得できる
	- `function`の返り値を`Response`とし、戻り値にレスポンスオブジェクトを指定する
