#Laravel 
# ルーティング状況の確認
```
php artisan route:list
//　または
sail artisan route:list
```
## 便利なオプション
```
# 特定のメソッドのみ表示
php artisan route:list --method=GET

# 特定の名前でフィルター
php artisan route:list --name=user

# 特定のパスでフィルター
php artisan route:list --path=api

# JSON形式で出力
php artisan route:list --json

# ミドルウェアも含めて詳細表示
php artisan route:list -v
```
