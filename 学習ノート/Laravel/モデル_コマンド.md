#Laravel 
# モデル作成
```
php artisan make:model [モデル名]
// または
sail artisan make:model [モデル名]
```
- 生成されるファイル
	- `app/Models/モデル名.php`

## マイグレーションファイルとセットで作成する場合
```
php artisan make:model [モデル名] -m
// または
sail artisan make:model [モデル名] -m
```
- 生成されるファイル
	- `app/Models/モデル名.php`
	- `database/migrations/マイグレーションファイル名.php`
